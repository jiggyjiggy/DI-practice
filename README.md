# DI-practice

- 한 object가 다른 ojbect를 사용할때 의존성이 있다고 말한다
- runtime 시 의존 관계를 맺는 대상을 외부에서 결정해서 주입하는 것을 "DI (Dependecy Injection; 의존성 주입)"라 한다
- spring framework는 DI 기능을 지원해준다
